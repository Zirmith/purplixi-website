Installation

Purplixi Launcher is available for Windows, macOS, and Linux. Follow these steps to get started.

System Requirements

Requirement Minimum Recommended
OS Windows 10, macOS 10.15, Ubuntu 18.04 Latest version
RAM 4 GB 8 GB+
Storage 500 MB (launcher only) 2 GB+
Java Java 8+ Java 17+

Installation Steps

  1. Download the installer for your operating system from the download page
  2. Run the installer and follow the on-screen instructions
  3. Launch Purplixi from your applications menu or desktop shortcut
  4. Sign in with your Microsoft account to start playing

On macOS, you may need to allow the app in System Preferences > Security & Privacy if you see a security warning.

First Setup

After installing Purplixi, follow these steps to configure your launcher for the best experience.

Initial Configuration

  1. Open Settings from the sidebar navigation
  2. Configure your Java settings - the launcher will auto-detect installed versions
  3. Set your preferred memory allocation (recommended: 4-8 GB)
  4. Choose your theme and appearance preferences
  5. Enable Discord Rich Presence if desired

Recommended Settings

For optimal performance, we recommend:

  • Java 17+ for Minecraft 1.18 and newer
  • 4-8 GB RAM allocation for vanilla Minecraft
  • 8-16 GB RAM for modded gameplay
  • Enable hardware acceleration

Authentication

Purplixi uses official Microsoft authentication to securely log into your Minecraft account.

Signing In

  1. Click "Add Account" on the home page or accounts section
  2. A browser window will open for Microsoft login
  3. Sign in with your Microsoft account credentials
  4. Grant permissions when prompted
  5. Return to the launcher - you're now logged in!

Multiple Accounts

Purplixi supports multiple Microsoft accounts. You can switch between accounts from the accounts page.

Never share your account credentials or auth tokens. Purplixi stores tokens securely and never transmits them to third parties.

Java Management

Purplixi includes advanced Java management features to help you configure the optimal Java setup for Minecraft.

Automatic Detection

The launcher automatically scans your system for Java installations including:

  • System Java (JAVA_HOME, PATH)
  • Downloads folder installations
  • Common installation directories
  • Version managers (SDKMAN, Jabba)

Custom Java Path

To use a specific Java installation:

/Users/username/Downloads/jdk-17.0.9/Contents/Home/bin/java

Java Version Requirements

Minecraft Version Required Java
1.20.5+ Java 21 (recommended)
1.18 - 1.20.4 Java 17+
1.17.x Java 16/17
1.16.5 and below Java 8

Memory Testing

Use the "Test Memory" button to verify that Java can allocate your requested RAM amount.

World Management

Manage your Minecraft worlds with import, backup, and restore functionality.

Importing Worlds

  1. Go to the Worlds tab
  2. Click "Import World"
  3. Select your world folder
  4. The world will be copied to your saves directory

Creating Backups

  1. Click "Create Backup" in the Worlds tab
  2. Select the world you want to backup
  3. The backup will be saved as a compressed ZIP file

Restoring Backups

Switch to the "Backups" tab to view and restore your saved backups. When restoring, your current world will automatically be backed up first for safety.

Backups are stored in the launcher's data directory and include timestamps for easy identification.

Server Browser

Browse and connect to popular Minecraft servers with real-time status information.

Featured Servers

The server browser includes 31+ popular servers across various categories:

  • Minigames: Hypixel, Mineplex, The Hive, etc.
  • Survival: EarthMC, 2b2t, etc.
  • Creative: Building servers
  • Factions: Competitive survival
  • Prison: Prison-themed gameplay
  • Skyblock: Island survival

Quick Connect

Click "Connect" on any server to automatically join after launching Minecraft.

Mod Support

Install and manage mods for Forge, Fabric, and other mod loaders.

Installing Mods

  1. Go to the Mods tab
  2. Click "Add Mod" or drag and drop JAR files
  3. The mod will be added to your mods folder
  4. Enable/disable mods as needed

Mod Compatibility

Ensure your mods are compatible with your Minecraft and mod loader versions. The launcher will display warnings for incompatible mods.

Only download mods from trusted sources like CurseForge or Modrinth. Never run mods from unknown sources.

JVM Arguments

Optimize Minecraft performance with custom JVM arguments.

Presets

Purplixi includes four built-in presets:

  • Default: No additional arguments
  • Performance: Optimized G1GC for best FPS
  • Low Memory: For systems with limited RAM
  • Modded: Optimized for modded Minecraft

Performance Arguments

-XX:+UseG1GC -XX:+ParallelRefProcEnabled -XX:MaxGCPauseMillis=200 -XX:+UnlockExperimentalVMOptions -XX:+DisableExplicitGC

Low Memory Arguments

-XX:+UseSerialGC -XX:+DisableExplicitGC -Xss512k

Frequently Asked Questions

Why won't Minecraft launch?

Check that you have the correct Java version installed for your Minecraft version. Use the Java Settings panel to validate your Java installation and test memory allocation.

How do I allocate more RAM?

Go to Settings > Java Settings and adjust the Maximum RAM slider. We recommend 4-8 GB for vanilla and 8-16 GB for modded.

Where are my worlds stored?

Worlds are stored in the standard Minecraft saves directory. Use the World Management feature to backup and manage your worlds.

How do I install Forge/Fabric?

Download and run the Forge or Fabric installer, then select the installed version in Purplixi's version selector.

Is my data safe?

Yes! Purplixi stores all data locally on your computer. Authentication tokens are encrypted and never shared with third parties.

How do I report a bug?

Visit our GitHub repository to report issues or join our Discord community for support.
