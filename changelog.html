November 20, 2025 Latest

Complete Server Management System

NEW: Full Server Management - Start, stop, restart, and delete servers with one click

Server Settings Editor - Modify port, max players, gamemode, difficulty, MOTD, and more
Beautiful Server Grid View showing all servers with live status indicators
Quick Folder Access from right-click context menu
View Server Logs from current session or previous runs
Error Highlighting in console with red text for easy debugging
Timestamped Logs with precise timestamps for every entry
Status Indicators showing live/offline states with real-time updates

NEW: Account Sorting - Sort by name (A-Z/Z-A) or date added (newest/oldest)
Export Accounts to JSON file (excluding sensitive tokens for security)
Bulk Token Refresh - Refresh all account tokens simultaneously
Quick Account Switching with seamless transitions
Visual indicators for active/inactive account status

Fixed "No handler registered for 'account-select'" error

Fixed "Create First Server" button not working
Server logs now display correctly in console view

Performance Improvements

Optimized server process management with efficient resource handling
Smart log storage - keeps last 1000 lines per server in memory
Real-time log streaming without constant polling for better performance
