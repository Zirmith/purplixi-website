Features

Everything You Need, and More

Packed with powerful features to enhance your Minecraft experience

Advanced Java Management

Automatic Java detection, version validation, memory testing, and smart recommendations. Supports custom paths and multiple installations with auto-scanning.

  • Auto-scan on startup
  • Search everywhere including Desktop
  • Deep recursive scanning
  • Fixed Mac Java extraction

World Management

Complete world control: edit properties, view players, track advancements, manage dimensions, and more. Full NBT support included.

  • Edit gamemode, difficulty, cheats
  • View players & advancements
  • Manage dimensions
  • Datapack guide & support

Microsoft Authentication

Secure Microsoft account integration with multi-account support. Switch between accounts seamlessly.

  • Official Microsoft login
  • Multiple accounts
  • Secure token storage
  • Auto-refresh tokens

Server Browser

Browse and connect to 31+ popular Minecraft servers. Live player counts and server status.

  • 31+ popular servers
  • Real-time status
  • Quick connect
  • Server categories

Mod Support

Install and manage mods effortlessly. Compatible with Forge, Fabric, and more.

  • Easy mod installation
  • Version compatibility
  • Enable/disable mods
  • Mod management

Beautiful Themes

Choose from 7 stunning themes with custom accent colors and backgrounds.

  • 7 beautiful themes
  • Custom backgrounds
  • Accent color options
  • Blur effects

Play Time Tracking

Track your gameplay sessions automatically. View total playtime, recent history, and version statistics.

  • Automatic session tracking
  • Accurate play time recording
  • Recent history display
  • Per-version statistics

Server Hosting

Host your own Minecraft servers with auto-download of server JARs (Vanilla & Paper), Java management, and live console.

  • Auto-download server JARs
  • Vanilla & Paper support
  • Auto Java detection
  • Live server console
